They are vector-based but for really crisp and pixel-perfect rendering, you are restricted to the specified font sizes. | | | | - | - | | Less-File | [icons.less](assets/less/core-elements/icons.less) | | Base CSS-Class | `.icon / .icon--*` | | Modifiers | `.x2, .x3, .x4, *-s` | ### Modifiers Modifiers are used to define the icon size. The initial size is 32px. `.x2` produces icons in 64px, `.x3` = 96px, `.x4` = 128px. In fact, every font-size as a multiple of 8px is possible for perfect rendering. For 16px, there a optimized versions with reduced detail. Simply use the postfix `*-s`. {% endmarkdown %}
