{% extends 'layouts/base.html' %} {% block main %} {% markdown %} ## Sticky Make any element on the page **stick** while scrolling past a certain threshold, which can be defined per element. | | | | - | - | | Js-File | {% if basePath == '/' %}[sticky.js](assets/js/modules/sticky.js){% else %}sticky.js{% endif %} | | Hooks | `data-sticky` | ### Simple Example Ad the custom data-attribute `data-sticky` to any element you wish to make *sticky*. While scrolling the page down, as soon as the vieport’s top border *touches* the element, it *sticks* at the viewport top while scrolling. If no other parameter is provided, the element *sticks as long as the page can be scrolled*. However, in this first simple example, one parameter is provided: *stop*. {% endmarkdown %}
{% markdown %} `data-sticky:stop:100;` from the example above tells the sticky element to stick as long, as it reached a position *100px further down from it’s initial position*. ### Picking a Reference Element A very useful feature of this script is the *reference element*. A valid reference element is any element on the page which is top-aligned with the sticky element and which is taller than the sticky element. You can provide a reference element by adding a CSS-selector as a value for the option *reference*: {% endmarkdown %} 
data-sticky="reference:#myReferenceElement;"
{% markdown %} The script selects the first matching element on the page and references it’s height to define a stop position for the sticky element. The sticky element *sticks* as long as it’s bottom aligns with the reference element’s bottom. {% endmarkdown %}
#myReferenceElement
{% markdown %} Another possible value for the reference-parameter is the keyword *parent*: {% endmarkdown %} 
data-sticky="reference:parent;"
{% markdown %} By using this keyword, the script selects the sticky element’s surrounding element (= parent element) as a reference. The sticky element sticks *inside it’s parent element* as long as the sticky element’s bottom reaches the parent element’s bottom. Please notice that the parent element’s bottom-padding is respected – if the parent element has a bottom-padding, the sticky element stops as soon as it’s bottom reaches the bottom padding. {% endmarkdown %}

What Is a Valid Reference Element?

A valid reference element is any element on the page which is top-aligned with the sticky element and taller than the sticky element.
