Markup

You can turn any HTML block-level element into a .posterTeaser but please note: only anchors get the hover effect.

Size

.posterTeaser__image sets the overall .posterTeaser width an height! Overflowing text will get cut off. Please also note, that .posterTeaser is set to inline-block. Wrap it inside another block-level element if you need to create a block.

{% markdown %} ### Modifiers #### posterTeaser\--bottom It makes the copy text align to the bottom. {% endmarkdown %}
