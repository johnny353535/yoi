scrollTo

This script provides a simple interface to set smooth automated scrolls to target anchors within the same document.
Js-File scrollTo.js
Hooks data-scrollto
Options highlight: [blink / pulse]

Smoothly Scroll to an Anchor

In order to set a smooth automated scroll (to a target element within the same document), add the data-attribute data-scrollto to the link.

Target #1

Highlighting Effects

If you wish to highlight the target element once the autometed scroll finished, add one of the two available highlight animation types – blink and pulse to the data-scrollto-attribute.

Scroll a Container

The target element may sit inside a container with hidden or scrollable overflow. In this case, instead of scrolling the whole page, only the container scrolls.

Positioning / Context

Please note: In order to scroll a container, the container itself must provide a context for positioning. This means it must be set to either position:relative, position:absolute or position:fixed.
