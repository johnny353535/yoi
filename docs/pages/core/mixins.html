Mixins

Mixins are reusable snippets of CSS code. They are a Less language feature.
Less-File mixins.less

Helper Mixins

.mixin-clearfix

Use this mixin to add the popular micro clearfix hack.

.mixin-hidden

Use this mixin to visually hide elements. These elements are still accessible by screenreaders.
