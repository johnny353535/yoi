Tables

Tables come with a few handy modifiers, eg. to render zebra-striped tables, tables with fixed width, etc. Tables can also be interactive (selectable or deletable rows), using the data-table hook.
Less-File tables.less
JS-File tables.js
Base CSS-Class none (just the table-tag) or .table
Modifiers .table--borderless, .table--colHeader, .table--compressed, .table--fixed, .table--simpleHeader, .table--stripped, .table-zebra

JavaScript Interface

data-table="
    removeable   : @param;
    selectable   : @param;
    btnModifiers : @param;
"
/**
 *  @param {bool}           removeable   - removeable table rows
 *  @param {bool || string} selectable   - if set to true – single table rows can be selected, if set to "multi" – multiple table rows can be selected
 */
