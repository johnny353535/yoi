Link List

Use this element to create a vertical menu-like list of links. Modifiers add options for different sizes and arrows.

Base CSS-Class .linkList
Modifiers .linkList__item--active, .linkList--large, .linkList--dark, .linkList--stripped, .linkList--transparent, .linkList--arrowsLeft, .linkList--arrowsRightUp, .linkList--arrowsRightDown
