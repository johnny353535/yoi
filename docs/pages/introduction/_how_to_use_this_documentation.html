How to Use This Documentation

This documentation was thoughtfully written and structured to make it easy to read and understand. It’s written in English – like all inline comments inside the code files.

Navigation

The documentation feature two menus. First, there is a large scrollable menu on the very right edge of the screen. It slides into view on mouse over. This is the main menu: it let’s you navigate all the pages of the documentation. A second menu can be found in the header. There are two prominent buttons that lead to the templates for the desktop and mobile version of the site.

Examples

Examples can be found on most pages. They can display both the rendered element and it’s markup. The code featues syntax-hightlighting for better readability.

Some examples have options to toggle the code block and to toggle the example background between light (default) and dark.
