Color Tool

Our color gradients were not created using Less’ internal color blending functions because they do not produce gradients with constant luminosity. Instead, certain shades would always pop out and feel wrong because they are over-saturated.

We solved this problem with a custom algorithm which basically mimics Photoshop’s »color« blend mode. Use the tool below to generate color shades. The color variables (less) for the Yoshino frontend are based on 25-step gradients. The higher the number, the lighter the color (eg. @blue-0 vs. @blue-25).
